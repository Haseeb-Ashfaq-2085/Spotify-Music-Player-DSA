# Spotify Music Player - DSA Project

## Overview
A backend-focused music player application that implements core Data Structures and Algorithms (DSA) to manage songs and playlists efficiently.

## Data Structures & Logic
* **Dynamic Array Management:** Implemented a custom dynamic array with a doubling-strategy for reallocation to handle growing playlists.
* **Efficient Deletion:** Algorithm logic to shift elements and fill gaps upon song removal to maintain list integrity.
* **Metadata Handling:** Structured storage for song titles, artists, and durations.
* **Searching & Sorting:** Optimized algorithms to filter and organize music libraries based on specific attributes.

## Key Features
* Create and manage multiple playlists.
* Add, remove, and search for songs.
* Simulated playback queue logic.
* Memory-efficient data handling using pointers and dynamic allocation.

## Tech Stack
* **Language:** C++
* **Concepts:** Dynamic Arrays, Pointers, Memory Management, and Search Algorithms.
